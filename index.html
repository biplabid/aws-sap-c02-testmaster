Import questions from a .docx file to expand your study materials.

File Format Instructions

To successfully import questions, your .docx file must follow a strict format. Each question must be separated by three dashes (---).

The structure for each question should be:

Question 1: [The question text...]
Type: [Single Choice/Multiple Choice]
Domain: [e.g., Design for New Solutions]
A. [Option A text]
B. [Option B text]
C. [Option C text]
D. [Option D text]
...
Answer: [Correct letter(s), e.g., A or A,C]
Explanation: [Detailed explanation text...]
---

Important: The file must contain at least 50 questions to be considered a valid question set.

After parsing, you'll be asked to pick this project's data folder so the new set can be saved there as the next available setN.json file. This requires a Chromium-based browser (Chrome or Edge).
